**S3-Bucket-Enumerator**
A simple S3 Bucket Enumerator Tool written in Python.

**Requirements:**

-Python 3.10+

-pip

-awscli

**Installation:**

git clone https://github.com/bounty-hunter235/S3-Bucket-Enumerator.git

cd S3-Bucket-Enumerator

pip install awscli

**Usage:**

python S3-Bucket-Enumerator.py
